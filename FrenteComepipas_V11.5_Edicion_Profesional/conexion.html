Conexión con Supabase

Configura y comprueba la conexión que usará el módulo de socios.

Estado

Pasos

  1. Ejecuta supabase/schema.sql.
  2. Edita assets/js/supabase-config.js.
  3. Activa enabled: true.
  4. Prueba la conexión y migra los datos.

Configuración

window.FRENTE_SUPABASE_CONFIG = {
  enabled: true,
  url: "https://...supabase.co",
  anonKey: "...",
  fallbackToLocal: true
};

Importante

No uses nunca la clave secreta service_role en una web pública. La protección depende de Auth y de las políticas RLS incluidas.
