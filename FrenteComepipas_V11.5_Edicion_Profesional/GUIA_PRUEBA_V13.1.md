# PRUEBA RÁPIDA V13.1

## Importar Excel
1. Abre `socios-admin.html`.
2. Pulsa `Importar Excel`.
3. Descarga la plantilla Excel.
4. Añade o cambia filas.
5. Sube el `.xlsx`.
6. Selecciona la hoja, si el archivo tiene varias.
7. Revisa columnas, errores y duplicados.
8. Aplica la importación.

## Exportar Excel
Prueba estos módulos:
- Socios → Exportar Excel.
- Renovaciones → Cobros → Exportar Excel.
- Material y solicitudes → Exportar Excel.

Las importaciones siguen guardándose localmente hasta conectar el módulo con Supabase.
