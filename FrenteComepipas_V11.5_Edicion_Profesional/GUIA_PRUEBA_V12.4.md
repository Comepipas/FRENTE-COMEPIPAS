# PRUEBA RÁPIDA V12.4

1. Abre `area-socio.html` con Go Live.
2. Entra en `Mi renovación`.
3. Pulsa `Confirmar que quiero renovar`.
4. Selecciona una forma de pago.
5. Pulsa `Simular pago`.
6. Comprueba el justificante.
7. Pulsa `Imprimir justificante` o `Reiniciar demostración`.

Todo se guarda únicamente en el navegador mediante localStorage.
No se realiza ningún cobro real.
