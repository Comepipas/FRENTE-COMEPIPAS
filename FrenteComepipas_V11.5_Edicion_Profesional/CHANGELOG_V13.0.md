# CHANGELOG V13.0

## Centro de importación de socios

- Nueva página `importacion-socios.html`.
- Carga mediante selector o arrastrar y soltar.
- Lectura de CSV con coma o punto y coma.
- Asignación automática y manual de columnas.
- Vista previa antes de guardar.
- Detección de:
  - socios nuevos;
  - socios existentes a actualizar;
  - números duplicados;
  - emails inválidos;
  - fechas con formato incorrecto.
- Importación parcial: las filas con error quedan fuera.
- Descarga de un CSV con los errores.
- Plantilla CSV incluida.
- Integración con la gestión local de socios.
