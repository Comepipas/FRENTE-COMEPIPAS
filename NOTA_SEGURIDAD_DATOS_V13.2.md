# Nota sobre los datos locales

Los datos guardados en `localStorage` pueden perderse al:

- borrar los datos del navegador;
- usar otro navegador;
- cambiar de ordenador;
- navegar en modo privado;
- restaurar el navegador.

Antes de hacer pruebas importantes, crea una copia desde:

`Administración → Copias de seguridad Excel`

La conexión permanente con Supabase seguirá siendo necesaria para producción.
